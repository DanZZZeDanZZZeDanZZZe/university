Изучение разметки табличных данных. Изучение CSS

Необходимо сверстать документ (HTML), который отображает таблицу:

  • Внешний вид должен соответствовать как в файле *.jpg.
  • Варианты заданий взять из соответствующей папки.
  • Таблица должна содержать данные для просмотра.
  • Необходимо вынести все стили (шрифт, цвет, стиль и т.п.), описывающие таблицу в отдельный CSS файл, и подключить его к разметке (HTML).

В финальном HTML-документе должна остаться только разметка докумен
